Deploy del proyecto Restack Media Plan en produccion en saulnoda.cloud/restack-media-plan

## Tareas Completadas

### 1. Build del proyecto
- Verificado vite.config.js con base path correcto (`/restack-media-plan/`)
- `npm install` - dependencias instaladas
- `npm run build` - build exitoso en dist/

### 2. Preparacion del servidor
- Conexion SSH a root@saulnoda.cloud
- Servidor: VPS Hostinger (Ubuntu)
- No habia servidor web instalado (solo Docker con otros proyectos)

### 3. Instalacion de Nginx
```bash
apt update && apt install -y nginx
```
- nginx 1.24.0 instalado y corriendo

### 4. Subida de archivos
- Creado directorio `/var/www/restack-media-plan/`
- Archivos subidos via SCP desde dist/

### 5. Configuracion Nginx
Archivo: `/etc/nginx/sites-available/saulnoda.cloud`
```nginx
server {
    listen 80;
    server_name saulnoda.cloud;

    location /restack-media-plan/ {
        alias /var/www/restack-media-plan/;
        try_files $uri $uri/ /restack-media-plan/index.html;
    }
}
```

### 6. SSL con Certbot
```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d saulnoda.cloud --non-interactive --agree-tos --email hola@saulnoda.com --redirect
```
- Certificado instalado exitosamente
- Redireccion HTTP -> HTTPS configurada
- Renovacion automatica habilitada
- Expiracion: 2026-04-03
